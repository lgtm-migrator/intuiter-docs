# Mouse Control
Win key is Meta key in Windows
- `win` + `ikjl` - mouse control with **accelleration**
- `Alt` + `uo` - mouse scroll with **acceleration**
- `Win` + `h;` - mouse scroll with **acceleration**
- `Win` + `uo` - Mouse left right click
