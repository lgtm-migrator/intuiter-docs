# Other features

## PC usage features
- `Win` + `s` - ask to Sleep Windows
- `Win` + `Shift` + `q` - ask to Shutdown Windows
- `Win` + `Alt` + `q` - Activate screen saver 
- `Win` + `Shift` + `l` - Lock Windows


## App usage features
- `Alt` + `yp` - Move among tabs instead of `ctrl` + `tab` with intuitive key alignment
