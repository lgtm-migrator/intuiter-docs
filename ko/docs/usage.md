## 주의사항

- 윈도우즈에서 여러 키보드 레이아웃을 사용한다면 아래 이미지처럼 `shift` +  `alt`을 이용하여 레이아웃을 바꾸는 단축키를 제거해야 프로그램을 원할하게 사용할 수 있습니다
 `win` + `space` 는 동일한 기능의 단축키입니다
![tray](/images/altshift.png)

- 이 프로그램은 게임 중에 핵으로 오해될 수 있으니 게임 중에는 프로그램 종료를 하는 걸 추천드립니다
